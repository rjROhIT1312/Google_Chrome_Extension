#

Usage
----------------

Clone -
```
$ git clone https://github.com/rjROhIT1312/Google_Chrome_Extension.git
```
Run -
```
1. open chrome://extensions/
2. Click on 'Load unpacked'
3. Select the directory with the source
```

The Jokes are all yours now! Enjoy:thumbsup:
